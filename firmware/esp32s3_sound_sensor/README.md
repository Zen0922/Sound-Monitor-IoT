# ESP32 Sound Sensor Firmware

ESP32 (XIAO ESP32S3) based sound level sensor.

## Hardware

- XIAO ESP32S3
- ICS-43434 MEMS microphone

## Function

- I2S microphone input
- RMS sound calculation
- Relative dB conversion
- BLE transmission
